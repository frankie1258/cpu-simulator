ขั้นตอนการประกอบ คอมพิวเตอร์

หลักการเลือก อุปกรณ์ คอมพิวเตอร์

ปัจจุบันแบ่งเป็น 2 แบรนด์หลักๆคือ intel และ AMD

GPU
( Graphics Processing Unit )

โปรเซสเซอร์ที่ออกแบบมาเป็นพิเศษเพื่อจัดการกับงานการเรนเดอร์กราฟิกที่หนักหน่วง กราฟิกที่สร้างด้วยคอมพิวเตอร์ เช่นที่พบในวิดีโอเกมหรือสื่อภาพเคลื่อนไหวอื่นๆนั้นถูกกำหนดให้แต่ละเฟรมแยกจากกันเพื่อให้สามารถ 'วาด' ทีละเฟรมได้โดยใช้คอมพิวเตอร์ ซึ่งจะต้องใช้พลังงานจำนวนมากกว่างานกราฟฟิคทั่วไป

MAINBOARD

แผงวงจรที่เชื่อมต่อฮาร์ดแวร์ของคุณทั้งหมดเข้ากับโปรเซสเซอร์ จ่ายไฟจากพาวเวอร์ซัพพลาย และกำหนดประเภทของอุปกรณ์จัดเก็บข้อมูล โมดูลหน่วยความจำ และการ์ดกราฟิกต่างๆ (ในการ์ดขยายอื่นๆ) ที่สามารถเชื่อมต่อกับพีซีของคุณได้

RAM
( Random-Access Memory )

หน่วยความจำการเข้าถึงแบบสุ่ม เป็นหน่วยความจำระยะสั้นของคอมพิวเตอร์ หรือแล็ปท็อป ทำหน้าที่เป็นที่จัดเก็บข้อมูลที่โปรเซสเซอร์คอมพิวเตอร์ของคุณต้องมี เพื่อเรียกใช้แอปพลิเคชันและเปิดไฟล์ต่าง ๆ ได้อย่างรวดเร็ว

HDD
( Hard Disk Drive )

จัดเก็บข้อมูลแบบถาวรในคอมพิวเตอร์. มันเป็นอุปกรณ์ที่ช่วยให้คุณสามารถเก็บไฟล์ รูปภาพ วิดีโอ เพลง เอกสาร และโปรแกรมต่าง ๆ ไว้ในระยะยาวได้. HDD ยังใช้จัดเก็บข้อมูลของระบบปฏิบัติการและไดรเวอร์ของคอมพิวเตอร์ด้วย

POWER SUPPLY

ทำหน้าที่แปลงไฟฟ้ากระแสสลับให้เป็นกระแสตรงเพื่อจ่าย ให้กับอุปกรณ์อื่น เช่น เมนบอร์ด ซีพียู แรม ฮาร์ดดิสก์ เป็นต้น

CASE

กล่องหรือโครงสร้างสำหรับเก็บประกอบอุปกรณ์ต่างๆ คอมพิวเตอร์ไว้ภายในนั้น ซึ่งขนาดของเคสก็จะแตกต่างกันออกไป แล้วแต่การใช้งานหรือความเหมาะสมในการใช้งานของแต่ละคนรวมทั้งสถานที่เก็บอุปกรณ์เหล่านั้นด้วยว่ามีขนาดพื้นที่มากน้อยเพียงใด

ขั้นตอนการประกอบ อุปกรณ์ คอมพิวเตอร์

วิธีประกอบคอมพิวเตอร์ด้วยตัวเองสำหรับมือใหม่

ขั้นตอนที่ 1: เลือกอุปกรณ์ให้เหมาะสมกับการใช้งาน

ก่อนจะเริ่มประกอบคอมคุณต้องรู้ว่าคอมพิวเตอร์ของคุณจะใช้ทำอะไร เช่น เล่นเกม ทำงานเอกสาร ตัดต่อวิดีโอ หรืองานกราฟิก แล้วเลือกสเปคที่เหมาะสมกับการใช้งาน โดยมีอุปกรณ์หลัก ๆ ดังนี้

  1. CPU (หน่วยประมวลผล): เป็นสมองของคอมพิวเตอร์ เลือกตามงบประมาณและการใช้งาน เช่น AMD Ryzen 5 หรือ Intel Core i5 สำหรับการเล่นเกมทั่วไป
  2. Mainboard (เมนบอร์ด): ต้องรองรับ CPU และ RAM ที่คุณเลือก เช่น เมนบอร์ดซ็อกเก็ต AM4 สำหรับ Ryzen หรือ LGA 1700 สำหรับ Intel
  3. RAM (หน่วยความจำ): ขนาด 16GB แบบ Dual Channel เหมาะสำหรับการเล่นเกมและทำงานทั่วไป
  4. GPU (การ์ดจอ): เลือกตามประเภทงาน เช่น การ์ดจอ NVIDIA RTX 3060 หรือ AMD RX 6600 XT สำหรับเกมและงานกราฟิก
  5. Storage (ที่เก็บข้อมูล): SSD ขนาด 512GB ขึ้นไป ช่วยให้เครื่องบูทเร็วและโหลดโปรแกรมได้ไวขึ้น
  6. Power Supply (PSU): เลือกกำลังไฟที่เหมาะสม เช่น 650W 80 Plus Bronze สำหรับสเปคกลาง
  7. Case (เคส): เลือกเคสที่มีพื้นที่พอสำหรับอุปกรณ์และมีระบบระบายความร้อนที่ดี
  8. Cooling System (ระบบระบายความร้อน): เลือกพัดลมหรือชุดน้ำปิดตามความต้องการของ CPU

 

ขั้นตอนที่ 2: เตรียมอุปกรณ์และเครื่องมือที่จำเป็น

การประกอบคอมให้ราบรื่น ควรเตรียมอุปกรณ์และเครื่องมือให้พร้อม ดังนี้

  • ไขควงหัวแฉกสำหรับขันน็อต
  • สายรัดเคเบิลสำหรับจัดระเบียบสายไฟ
  • ถุงมือป้องกันไฟฟ้าสถิต (ESD Gloves) เพื่อป้องกันไฟฟ้าสถิตทำลายอุปกรณ์
  • คู่มืออุปกรณ์ เช่น คู่มือเมนบอร์ดและเคส

 

ขั้นตอนที่ 3: การติดตั้งฮาร์ดแวร์ในเคส

1. ติดตั้ง CPU บนเมนบอร์ด

  • เปิดช่องซ็อกเก็ต CPU บนเมนบอร์ด
  • วาง CPU ลงไปอย่างระมัดระวังให้ตรงกับขาและลูกศรบนเมนบอร์ด
  • ปิดช่องซ็อกเก็ตและล็อกให้เรียบร้อย

2. ติดตั้ง RAM บนเมนบอร์ด

  • เปิดช่องใส่ RAM บนเมนบอร์ด
  • ใส่ RAM ลงในช่องให้แน่นจนได้ยินเสียงคลิก ควรติดตั้งในช่องสลับกัน (Dual Channel)

3. ติดตั้ง M.2 SSD (ถ้ามี)

  • เสียบ SSD ลงในช่อง M.2 บนเมนบอร์ด แล้วขันน็อตล็อกให้เรียบร้อย

4. ติดตั้งเมนบอร์ดในเคส

  • วางเมนบอร์ดลงในเคสและขันน็อตยึดให้แน่น
  • ตรวจสอบว่าไม่มีการขันน็อตแน่นเกินไปจนเมนบอร์ดเสียหาย

5. ติดตั้ง Power Supply (PSU)

  • ใส่ PSU ลงในช่องที่กำหนดไว้ในเคส
  • ยึด PSU ด้วยน็อตให้แน่น แล้วเชื่อมต่อสายไฟหลัก เช่น 24-pin ATX และ 8-pin CPU

6. ติดตั้งการ์ดจอ (GPU)

  • เสียบการ์ดจอลงในสล็อต PCIe บนเมนบอร์ด
  • ขันน็อตยึดการ์ดจอเข้ากับเคสให้แน่น และเชื่อมต่อสายไฟ PCIe จาก PSU

7. จัดสายไฟภายในเคส

  • จัดระเบียบสายไฟให้เรียบร้อยเพื่อให้การไหลเวียนอากาศดีขึ้น และปิดเคสได้อย่างสวยงาม

 

ขั้นตอนที่ 4: ตรวจสอบและบูทเครื่องครั้งแรก

หลังจากประกอบคอมเสร็จเรียบร้อย ให้ทำการตรวจสอบดังนี้

  1. ตรวจสอบการเชื่อมต่อ: สายไฟทุกเส้นต้องเชื่อมต่อกับอุปกรณ์อย่างถูกต้อง
  2. เสียบสายจอมอนิเตอร์และคีย์บอร์ด: เชื่อมต่ออุปกรณ์ภายนอก เช่น จอภาพ เมาส์ และคีย์บอร์ด
  3. เปิดเครื่อง: กดปุ่มเปิดเครื่อง และเช็คว่าพัดลม CPU, GPU และ PSU ทำงานปกติ
  4. เข้า BIOS: กดปุ่ม Del หรือ F2 เพื่อตั้งค่า BIOS ตรวจสอบการตรวจพบฮาร์ดแวร์ เช่น CPU, RAM และ Storage
  5. ติดตั้งระบบปฏิบัติการ: ใช้แฟลชไดร์ฟติดตั้ง Windows หรือระบบปฏิบัติการอื่น ๆ

 

ข้อผิดพลาดที่ควรหลีกเลี่ยงในการประกอบคอม

  1. ไม่ติดตั้งซิลิโคนบน CPU: ทำให้ระบบระบายความร้อนไม่มีประสิทธิภาพ
  2. ขันน็อตแน่นเกินไป: อาจทำให้เมนบอร์ดเสียหาย
  3. เสียบ RAM ไม่แน่น: เครื่องจะไม่บูทหรือเกิดปัญหา
  4. เสียบสายไฟไม่ครบ: เช่น ลืมเสียบสาย CPU 8-pin หรือ GPU PCIe
  5. ระบบระบายความร้อนไม่ดี: ทำให้อุณหภูมิสูงเกินไป ส่งผลต่ออายุการใช้งาน
