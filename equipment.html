อุปกรณ์ คอมพิวเตอร์

ทำความรู้จักกับ อุปกรณ์ คอมพิวเตอร์

CPU
( Central Processing Unit )

หน่วยประมวลผลกลางของคอมพิวเตอร์ เปรียบเสมือนสมองของมนุษย์ ทำหน้าที่ประมวลผลเชิงตรรกะ ผ่านกระบวนการ อ่าน ตีความ และประมวลผลชุดคำสั่ง นอกจากนี้ ซีพียูยังทำหน้าที่อ่านข้อมูลจากหน่วยความทรงจำ เขียนข้อมูล ส่งผลการประมวลผล และควบคุมการทำงานของอุปกรณ์ทั้งหมดของคอมพิวเตอร์รวมทั้งอุปกรณ์อื่น ๆ ที่ต่อพ่วงเข้ากับคอมพิวเตอร์ด้วย โดยมีหน่วยวัดความเร็วเป็นเมกะเฮริตซ์ (MHz) จนถึงกิกะเฮริตซ์ (GHz)

GPU
( Graphics Processing Unit )

โปรเซสเซอร์ที่ออกแบบมาเป็นพิเศษเพื่อจัดการกับงานการเรนเดอร์กราฟิกที่หนักหน่วง กราฟิกที่สร้างด้วยคอมพิวเตอร์ เช่นที่พบในวิดีโอเกมหรือสื่อภาพเคลื่อนไหวอื่นๆนั้นถูกกำหนดให้แต่ละเฟรมแยกจากกันเพื่อให้สามารถ 'วาด' ทีละเฟรมได้โดยใช้คอมพิวเตอร์ ซึ่งจะต้องใช้พลังงานจำนวนมากกว่างานกราฟฟิคทั่วไป

MAINBOARD

แผงวงจรที่เชื่อมต่อฮาร์ดแวร์ของคุณทั้งหมดเข้ากับโปรเซสเซอร์ จ่ายไฟจากพาวเวอร์ซัพพลาย และกำหนดประเภทของอุปกรณ์จัดเก็บข้อมูล โมดูลหน่วยความจำ และการ์ดกราฟิกต่างๆ (ในการ์ดขยายอื่นๆ) ที่สามารถเชื่อมต่อกับพีซีของคุณได้

RAM
( Random-Access Memory )

หน่วยความจำการเข้าถึงแบบสุ่ม เป็นหน่วยความจำระยะสั้นของคอมพิวเตอร์ หรือแล็ปท็อป ทำหน้าที่เป็นที่จัดเก็บข้อมูลที่โปรเซสเซอร์คอมพิวเตอร์ของคุณต้องมี เพื่อเรียกใช้แอปพลิเคชันและเปิดไฟล์ต่าง ๆ ได้อย่างรวดเร็ว

HDD
( Hard Disk Drive )

จัดเก็บข้อมูลแบบถาวรในคอมพิวเตอร์. มันเป็นอุปกรณ์ที่ช่วยให้คุณสามารถเก็บไฟล์ รูปภาพ วิดีโอ เพลง เอกสาร และโปรแกรมต่าง ๆ ไว้ในระยะยาวได้. HDD ยังใช้จัดเก็บข้อมูลของระบบปฏิบัติการและไดรเวอร์ของคอมพิวเตอร์ด้วย

SSD
( Solid State Drive )

เป็นอุปกรณ์จัดเก็บข้อมูลถาวรในคอมพิวเตอร์และอุปกรณ์อื่นๆ โดยใช้หน่วยความจำแบบ Flash Memory แทนจานหมุนเหมือนฮาร์ดดิสก์ ข้อดีคือทำงานได้เร็วกว่า ทนทานกว่า กินไฟน้อยกว่า และเงียบกว่าฮาร์ดดิสก์

M.2

หนึ่งในรูปแบบของ SSD ซึ่ง M.2 ก็คือ SSD ที่ได้รับการออกแบบมาใหม่ ทำให้การเชื่อมต่อมีรูปแบบใหม่ และมีประสิทธิภาพที่สูงขึ้นกว่าแบบเดิมอย่าง SATA ที่เราคุ้นเคย แถมยังเพิ่มความแรง มาในดีไซน์ที่เล็กลงอีกด้วย ดังนั้น SSD กับ M.2 คือหน่วยความจำเหมือนกัน เพียงแต่ M.2 เป็นการพัฒนาการเชื่อมต่อกับเมนบอร์ดและอุปกรณ์ต่าง ๆ ขึ้นมาใหม่ และกำลังได้รับความนิยมในการใช้งานมากขึ้นเรื่อย ๆ อีกด้วย

POWER SUPPLY

ทำหน้าที่แปลงไฟฟ้ากระแสสลับให้เป็นกระแสตรงเพื่อจ่าย ให้กับอุปกรณ์อื่น เช่น เมนบอร์ด ซีพียู แรม ฮาร์ดดิสก์ เป็นต้น

HEATSINK

อุปกรณ์ที่ออกแบบมาเพื่อ ทำหน้าที่ลดอุณหภูมิขณะทำงานของอุปกรณ์คอมพิวเตอร์หรืออุปกรณ์เอิเล็กทรอนิกส์ โดยเพิ่มพื้นที่สัมผัสอากาศ ทำให้การพาความร้อนจากตัวอุปกรณ์สู่อากาศโดยรอบทำได้เร็วขึ้น

LIQUID

อุปกรณ์ที่ระบายความร้อนด้วยของเหลวใช้สารหล่อเย็นเพื่อถ่ายเทความร้อนจากซีพียู

CASE

กล่องหรือโครงสร้างสำหรับเก็บประกอบอุปกรณ์ต่างๆ คอมพิวเตอร์ไว้ภายในนั้น ซึ่งขนาดของเคสก็จะแตกต่างกันออกไป แล้วแต่การใช้งานหรือความเหมาะสมในการใช้งานของแต่ละคนรวมทั้งสถานที่เก็บอุปกรณ์เหล่านั้นด้วยว่ามีขนาดพื้นที่มากน้อยเพียงใด

KEYBOARD

อุปกรณ์ที่ใช้นำข้อมูลจากผู้ใช้เข้าสู่เครื่องคอมพิวเตอร์หรือเครื่องมืออิเล็กทรอนิกส์ มีลักษณะเป็นแป้นสี่เหลี่ยม ด้านในมีปุ่มกดเป็นทั้งตัวอักษร ตัวเลข หรือสัญลักษณ์ ซึ่งสามารถป้อนข้อมูลได้หลายภาษา ขึ้นอยู่กับการติดตั้งและเพิ่มภาษานั้นๆ

MOUSE

อุปกรณ์ที่ทำหน้าที่ป้อนข้อมูลอย่างหนึ่งแต่ที่เห็นการทำงานโดยทั่วไปจะเป็นตัวที่ใช้ควบคุมลูกศรให้เคลื่อนที่ไปยังตำแหน่งต่างๆ บนจอภาพ เหมาะสำหรับใช้งานเมื่อต้องเลือก หรือเลื่อนวัตถุต่างๆ บนจอ

MONITOR

อุปกรณ์คอมพิวเตอร์ซึ่งทำหน้าที่แสดงผลลัพธ์ที่ได้จากการประมวลผลของคอมพิวเตอร์ โดยรับข้อมูลแล้วแปลงออกมาเป็นสัญญาณภาพ ไม่ว่าจะเป็นภาพ ตัวอักษร ตัวเลข สัญลักษณ์ รวมไปถึงสีสันต่าง ๆ ด้วย ซึ่งผลลัพธ์ที่ถูกแสดงออกมานั้นจะเปลี่ยนแปลงไปตามข้อมูลที่คอมพิวเตอร์ได้รับนั่นเอง
